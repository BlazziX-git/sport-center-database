# Sport Center Database

База данных информационной системы спортивного центра.

## СУБД
MySQL 8.0

## Описание
Проект содержит SQL-дамп структуры и данных базы данных,
используемой в курсовой работе по дисциплине «Базы данных».

## Содержимое репозитория
- sport_center_db.sql — структура и данные базы данных
